# YigitEfeTuglu_251109024_WebProjesi

Bu sene ders kapsamında geliştirdiğim doğa gezi yönetim sistemi projesidir. Gezilecek yerleri listelemek, rezervasyon almak ve yönetici paneli üzerinden verileri yönetmek üzerine kurulu bir çalışma.

Neler Kullandim? Frontend: HTML, CSS, JavaScript (Fetch ile dinamik veri çekimi)

Backend: PHP, MySQL (PDO yapısı ile)

Neler Yaptim? Yönetici paneli ve kullanıcı girişi için session yapısını kullandım.

SQL Injection riskine karşı veritabanı sorgularını prepare/execute yöntemiyle korumaya aldım.

Gezi listeleme ve rezervasyon kayıt işlemlerini temel seviyede sorunsuz çalışacak şekilde bağladım.

Geliştirme aşamasında hataları takip edebilmek için hata raporlama komutlarını kullandım.

Proje XAMPP (Apache ve MySQL) üzerinde çalışacak şekilde tasarlandı:

Projeyi bilgisayarınıza indirin veya klonlayın.
Klasörü C:\xampp\htdocs\ dizinine yerleştirin.
XAMPP panelinden Apache ve MySQL'i başlatın.
http://localhost/phpmyadmin/ adresine girip 251109024_webproje_doga adında bir veritabanı oluşturun.
İçerideki 251109024_webproje_doga.sql dosyasını bu veritabanına "İçe Aktar" diyerek yükleyin.
Tarayıcıdan http://localhost/YigitEfeTuglu_251109024_WebProjesi/ adresine giderek projeyi çalıştırın.
